① 素のJavaScriptで最小カルーセル（学習用）

仕組みの肝を 30 行ちょいで実装。translateX で帯を動かすだけ。

Slide 2
Slide 3
<div class="plain-carousel">
  <div class="plain-track">
    <div class="plain-slide">...</div>
    <div class="plain-slide">...</div>
  </div>
</div>
<script>
const track = document.querySelector('.plain-track');
const slides = [...document.querySelectorAll('.plain-slide')];
let i = 0;
function go(dir){
  i = (i + dir + slides.length) % slides.length;
  track.style.transform = `translateX(${-100 * i}%)`;
}
// 初期化＆操作
track.style.width = `${slides.length * 100}%`;
slides.forEach(s => s.style.width = `${100 / slides.length}%`);
document.querySelectorAll('.plain-btn').forEach(b=> b.addEventListener('click', ()=> go(+b.dataset.dir)));
</script>

※ 無限ループやドラッグは省略。原理理解用の最小実装です。

④ 比較とハマりポイント

  • Slick: jQuery必須。既存資産に乗せやすい。（CDNは jQuery → Slick の順）
  • Swiper: 依存なし。タッチ＆GPUでヌルヌル。
  • 共通: 親要素が display:none の状態で初期化しない（幅が計算できない）。
  • 画像比率バラバラ問題は CSS で object-fit: cover を付ける。
  • アクセシビリティ: キーボード操作有効化、aria-label/roledescription の付与。
