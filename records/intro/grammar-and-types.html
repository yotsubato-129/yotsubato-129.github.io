文法とデータ型

JavaScriptの基本文法とデータ型について学んだ内容を詳細にまとめています。

基本

  • JavaScriptはUnicode 文字セットを使用する
  • 大文字と小文字を区別する
  • 命令は文 (statement) と呼ばれ、セミコロン (;) によって区切られる

コメント

コメントはスクリプトの実行から除外され、ホワイトスペースのように扱われます。

行コメント

// 一行のコメントは先頭に記号を付ける

ブロックコメント

/*
複数行からなる
コメントを囲む
*/

ブロックコメントの注意点は、入れ子のコメントができずSyntaxErrorとなる点です。

変数と宣言

変数はデータを格納し参照するための識別子で、文字、アンダースコア (_)、ドル記号 ($) から始めます。

変数宣言の方法

  • var：ローカル変数とグローバル変数の両方に使用
  • let：ブロックスコープのローカル変数に使用
  • const：ブロックスコープで読み取り専用の定数を宣言

データ型

JavaScriptには以下のデータ型が存在します。

  • Boolean（論理値）：true または false
  • null：null 値を意味する特殊なキーワード
  • undefined：未定義の最上位プロパティ
  • Number：整数または浮動小数点数
  • BigInt：精度が自由な整数値
  • String（文字列）：テキストの値を表す連続した文字
  • Symbol：インスタンスが固有で不変となるデータ型
  • Object：データとそのデータをやり取りする命令を持つデータ構造

データ型の変換

JavaScriptではデータ型の自動変換が行われます。例えば、数値と文字列を + 演算子で結合すると、数値が文字列に変換されます。

const x = "答えは " + 1 + 2;
console.log(x); // "答えは 12"

文字列から数値への変換も自動で行われる場合があります。
